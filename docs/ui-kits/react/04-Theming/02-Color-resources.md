# **Color Resources**

## Introduction

The **Chat UI Kit** features a carefully crafted **color palette** designed for a **consistent and visually appealing** user experience. It follows the **Block, Element, Modifier (BEM)** methodology, ensuring **scalable styling** and easy **customization** by overriding the Kitâ€™s CSS variables.

---

## Color Palette

The **primary color** defines key actions, branding, and UI elements, while the **extended primary palette** provides variations for supporting components.
